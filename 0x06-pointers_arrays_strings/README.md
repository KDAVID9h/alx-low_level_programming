# Projet C - Manipulation de pointeurs, tableaux et chaînes de caractères

Ce dossier contient les fichiers sources et les informations spécifiques au projet de manipulation de pointeurs, de tableaux et de chaînes de caractères en langage C.

## Contenu

Ce dossier comprend les fichiers sources pour les tâches du projet, ainsi qu'un fichier d'en-tête `main.h`. Voici la liste des fichiers inclus :

- `0-strcat.c`: Fonction pour concaténer deux chaînes de caractères.
- `1-strncat.c`: Fonction pour concaténer deux chaînes de caractères jusqu'à un nombre donné de caractères.
- `2-strncpy.c`: Fonction pour copier une chaîne de caractères jusqu'à un nombre donné de caractères.
- `3-strcmp.c`: Fonction pour comparer deux chaînes de caractères.
- `4-rev_array.c`: Fonction pour inverser le contenu d'un tableau d'entiers.
- `5-string_toupper.c`: Fonction pour convertir les lettres minuscules d'une chaîne en majuscules.
- `6-cap_string.c`: Fonction pour capitaliser chaque mot d'une chaîne de caractères.
- `7-leet.c`: Fonction pour encoder une chaîne de caractères en "leet speak".
- `main.h`: Fichier d'en-tête contenant les prototypes de toutes les fonctions.

## Exécution

Chaque tâche peut être compilée et exécutée individuellement. Consultez le README.md général à la racine du dépôt pour obtenir des instructions détaillées sur la compilation et l'exécution de chaque tâche.
